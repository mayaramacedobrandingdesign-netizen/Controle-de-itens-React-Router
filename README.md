# Controle de Itens com React

Este projeto foi desenvolvido utilizando **React** e **React Router** para praticar a criação de um sistema simples de **CRUD** (Create, Read, Update e Delete).

A aplicação permite adicionar itens com quantidade, visualizar a lista de itens cadastrados, editar informações e remover itens. Caso um item seja adicionado novamente, o sistema soma automaticamente a quantidade ao item existente.

## Projeto online

https://controle-de-itens-react-router.vercel.app/

## Tecnologias utilizadas

- React
- React Router DOM
- JavaScript
- CSS
- Vite

## Funcionalidades

- Adicionar itens com controle de quantidade
- Listar itens em formato de cards
- Editar nome e quantidade dos itens
- Remover itens da lista
- Navegação entre páginas sem recarregar a aplicação
